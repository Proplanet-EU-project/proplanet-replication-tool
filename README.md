## Project Installation
To install the project you must make sure you have docker installed in your computer, Visual Studio is updated and .NET SDK is at the latest version

### Install Docker
To install Docker, you need to download it from the official Docker website ([download link](https://www.docker.com/products/docker-desktop/)). Click on "Download for Windows" and follow the installation process.

### Update Visual Studio
   1. Press Windows key
   2. Write Visual Studio Installer
   3. Click on the option with the same name
   4. A window should appear
   5. Click on update all

### Update .NET SDK
To update the .NET SDK, you can visit the [Microsoft .NET Download page](https://dotnet.microsoft.com/download) and follow the instructions provided.
